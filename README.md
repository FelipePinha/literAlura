# LiterAlura

## Detalhes do projeto 

LiterAlura é um desafio proposto no curso ONE da Alura. 

A aplicação consiste em consumir a API do Gutendex, salvar os dados de autores e livros no banco e manipular esses dados.

## Técnologias usadas

- Java
- Spring Boot
